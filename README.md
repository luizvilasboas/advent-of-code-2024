# advent-of-code-2024

> My solutions for the Advent of Code 2024 challenges, written in Perl.

## About the Project

This repository contains my personal solutions for the [Advent of Code 2024](https://adventofcode.com/2024) programming puzzles. Each day's solution is organized within its respective directory.

## Tech Stack

*   [Perl](https://www.perl.org/)

## Usage

To run a solution for a specific day, navigate to the corresponding directory and execute the Perl script.

```bash
# Example for day 1
cd day_1
perl solution.pl
```
